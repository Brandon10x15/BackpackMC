# BackpackMC
- Paper 1.21.1, Java 21
- Permissions-based backpack rows (1-6)
- Storage backends: YAML, SQLite, MySQL
- Auto-pickup to backpack when inventory is full
- Shortcut item to open backpack
- Item filter, multi-language, keep-on-death handling
- Commands: /backpack, /clearinventory
